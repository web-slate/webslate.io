What is Queue

  1. Queue data structure follows FIFO(First In, First Out) principle.
  2. A Queue is a container where only the front and back elements can be accessed or operated upon.
  3. Elements can be added only from back (rear) and Elements can be removed only from front.
  4. Enqueue in back (tail) and Dequeue in Front (head).

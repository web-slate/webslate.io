What is Stack

  1. Stack is a linear data structure to store a one-dimensional list.
  2. Store items in Last-in, First-Out or First-In, Last-Out(FILO).
  3. Elements can be added to or removed from a stack at only from top.

When do we use Stack

  1. In Browser, When user wants to go (history) Back/Forward (stacks) in a web browser.
  2. In Editors / Microsoft Word, When user wants to do Undo or Redo Operation.
  3. In Programming Language, Expression evaluation and syntax parsing.
  4. In Virtual Machines, example JVM are stack oriented.
